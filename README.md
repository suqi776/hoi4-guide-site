### Hearts of Iron 4 Strategy Guide

这是一个专门为《钢铁雄心4》（Hearts of Iron 4，以下简称HOI4）打造的中文战略指南网站。我们致力于为玩家提供全面、深入的游戏攻略和教程，帮助大家更好地理解和享受这款战略模拟游戏。

#### 主要内容

- **国家攻略**: 详细分析各大国（德国、苏联、美国等）的开局策略、军事建设、经济规划、外交政策等
- **基础教程**: 为新手玩家提供游戏机制介绍和基本玩法指导
- **高级策略**: 探讨多人游戏玩法、外交技巧、间谍活动、核战略等进阶内容
- **Mod推荐**: 热门模组介绍和安装指导

#### 特色功能

- 📚 **目录导航**: 使用markdown-it-table-of-contents自动生成页面目录，便于快速跳转
- 🌓 **主题切换**: 支持明暗主题自动适应
- 📱 **响应式设计**: 完美适配手机和桌面端
- 🚀 **现代化技术栈**: 基于Nuxt 4 + TypeScript构建

### 技术栈

本项目使用了现代化的Web开发技术：

- **框架**: Nuxt 4 (Vue 3 + TypeScript)
- **CSS**: UnoCSS (原子化CSS引擎)
- **状态管理**: Pinia
- **UI组件**: VueUse 组合API
- **Markdown渲染**: Markstream-vue
- **构建工具**: pnpm + Vite

### 部署运行

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 生产构建
pnpm build

# 预览构建结果
pnpm preview
```
